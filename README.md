# backend-test
Backend Test Project

## Install
`$ npm install`

## Run Dev
`$ npm run dev` 

## Run
`$ npm run start` 

## Config
file **.env**

>HOST=localhost **//Database Host**
>
>USER=root **//Database User**
>
>DB=db_test_baania **//Database Name**
>
>DIALECT=postgres 
>
>PORT=8000 **//API  Port**
